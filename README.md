# CSS Playgrounds

Sandbox to play with different CSS Frameworks.

### Frameworks Supported

- [x] Vanilla
- [x] SCSS
- [x] Bootstrap
- [x] Bulma
- [x] Tailwind
- [x] Materialize

For any other CSS Framework support, kindly create an issue.

### Features

- [ ] Download Code Zip
- [x] Local Code Saving
- [ ] Code Sharing
